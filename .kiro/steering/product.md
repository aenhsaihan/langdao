# Product Overview

LangDAO is a Web3-enabled language tutoring platform where people use their language skills to teach and earn cryptocurrency.

## Core Value Proposition

- **For Students**: Connect with native speakers for live language practice sessions, pay only for time used
- **For Tutors**: Monetize language skills by teaching, earn crypto (PYUSD) per second of tutoring
- **For Web3 Ecosystem**: Human-powered onboarding and language access for global adoption

## Key Features

- **Real-time Matching**: Socket.io-based instant pairing of students with available tutors
- **Live Video Sessions**: Custom WebRTC implementation for peer-to-peer video calls
- **Crypto Payments**: Timestamp-based payment system using PYUSD on Sepolia testnet
- **Wallet-First**: All interactions require wallet connection (MetaMask, WalletConnect, etc.)
